# SaaS Startup Landing Page

A modern, responsive **SaaS product landing page** created for the **Cognetix Technology – Front End Developer Internship (Foundational Stage | Task 2)**.

---

## Project Overview

This project is a **one-page SaaS landing website** designed to highlight a startup’s product, features, pricing plans, and customer testimonials. The page follows modern UI/UX principles and is fully responsive on all devices.

---

## Objective

* Design a professional **startup/product landing page**
* Use basic front-end skills with **HTML, CSS, and JavaScript**
* Create a **mobile-first, responsive UI**
* Maintain a clean project structure and deployment

---

## Technologies Used

* **HTML5** – Semantic structure
* **CSS3** – Flexbox, Grid, responsive layouts
* **JavaScript (ES6)** – Navbar toggle and interactivity
* **Google Fonts** – Typography
* **SVG Icons and Illustrations** – UI visuals

---

## Features

* Sticky navigation bar
* Hero section with SaaS illustration and call-to-action
* Feature cards with icons
* Pricing plans with highlighted recommendation
* User testimonials with avatars
* Final call-to-action section
* Smooth scrolling
* Fully responsive (mobile, tablet, desktop)
